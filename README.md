# 🎓 Student Portal - Web Engineering Lab

This is a comprehensive, responsive Student Management Portal developed as part of a Web Engineering Lab project. The application features a clean UI and handles core functionalities like registration, course selection, and user feedback.

## 🚀 Key Features

* **Landing Page (`index.html`):** A welcoming entry point with a unified navigation bar for easy access to all modules.
* **Student Registration:** A complete form with validation fields for name, email, password, age, and gender.
* **Interactive Dashboard:** A central hub where students can view their status and check the current date and time dynamically.
* **Course Selection:** A dedicated module for students to select from various subjects like Web Dev, AI, Database, and Mobile Apps.
* **Feedback System:** A form-based system for users to provide names, comments, and ratings (Excellent, Good, Average).

## 🛠️ Technologies Used

* **HTML5:** Used for building the structure of all web pages.
* **Bootstrap 5:** Utilized for modern, responsive layouts and pre-styled components like buttons and navbars.
* **CSS3:** Custom styles integrated via `style.css` for personalized branding.
* **JavaScript:** Powering dynamic features like form validation (`validateForm`), course submission (`submitCourses`), and date display (`showDateTime`).

## 📂 File Structure

* `index.html`: The main homepage.
* `registration.html`: Student sign-up page.
* `dashboard.html`: The main user interface after login.
* `courses.html`: Subject selection page.
* `feedback.html`: User review and rating page.
* `style.css`: External stylesheet for UI design.
* `script.js`: Logic for interactive features and form handling.

## 🌐 Deployment

This project is ready to be hosted on **GitHub Pages**. To view it live:
1. Go to your Repository **Settings**.
2. Select **Pages** from the sidebar.
3. Set the branch to `main` and click **Save**.
